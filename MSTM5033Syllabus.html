Course Description

This course introduces fundamental concepts of data communication and networking, such as network structure, cybersecurity issues, and trends in communications and networking. Practical application of content is made through case study analysis.

Course Materials

  • Textbook: West, J., & White, C. M. (2023). *Data Communication and Computer Networks* (9th ed.). Cengage.
  • Software: MS Office, Packet Tracer

Learning Outcomes

  1. Apply fundamental concepts of data communication and networking with practical applications
  2. Examine the role of network layers
  3. Evaluate cybersecurity issues
  4. Analyze key trends in communications and networking
